# Competency centre
Epam test

##How to build project

Clone a project:

```bash
git clone git@github.com:mrsecret911/competency-centre.git
```

Enter to the folder:

```bash
cd competency-centre
```

Run npm install:

```bash
npm install
```

Run bower:

```bash
bower update
```

To build project use:

```bash
gulp
```

Run server:

```bash
gulp serve
```
Open link http://localhost:8080
